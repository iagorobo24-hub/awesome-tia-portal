# Bloques de función

Aquí viven los **Function Blocks (FB)** y **Functions (FC)** exportados desde TIA Portal.

- **FB** → tiene memoria interna, necesita bloque de datos de instancia
- **FC** → sin memoria, solo entradas/salidas

Cada recurso tiene su propia subcarpeta con el `.xml` exportado y un `README.md` que explica qué hace y cómo usarlo.
