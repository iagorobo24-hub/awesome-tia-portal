# Tipos de datos (UDT)

**User Defined Types** reutilizables — estructuras de datos personalizadas para organizar variables de forma limpia y consistente entre proyectos.

Ejemplos típicos: estructura de un motor, de un sensor analógico, de una válvula, de un eje...
