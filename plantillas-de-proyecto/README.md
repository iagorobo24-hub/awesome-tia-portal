# Plantillas de proyecto

Estructuras base de proyecto TIA Portal — ya con la organización de carpetas, convenciones de nombres y bloques vacíos listos para rellenar.

El objetivo es tener un punto de partida limpio para cada tipo de proyecto (pequeña máquina, línea con varios PLCs, proyecto con HMI...) sin perder tiempo montando la estructura desde cero.
